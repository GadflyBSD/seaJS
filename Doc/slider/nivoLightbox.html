nivo Lightbox灯箱插件

关于

jQuery nivo Lightbox项目主页：http://dev7studios.com/plugins/nivo-Lightbox/ 版本号：jQuery Nivo Lightbox v1.1

通过 data-option或者 data-config，设定参数

  • effect参数，默认值：fade The effect to use when showing the lightbox

  • theme参数，默认值：default The lightbox theme to use

  • keyboardNav参数，默认值：true Enable/Disable keyboard navigation (left/right/escape)

  • clickOverlayToClose参数，默认值：true If false clicking the "close" button will be the only way to close the lightbox

  • onInit参数，默认值：function(){} Callback when lightbox has loaded

  • beforeShowLightbox参数，默认值：function(){} Callback before the lightbox is shown

  • afterShowLightbox参数，默认值：function(lightbox){} Callback after the lightbox is shown

  • beforeHideLightbox参数，默认值：function(){} Callback before the lightbox is hidden

  • afterHideLightbox参数，默认值：function(){} Callback after the lightbox is hidden

  • onPrev参数，默认值：function(){} Callback when the lightbox gallery goes to previous item

  • onNext参数，默认值：function(){} Callback when the lightbox gallery goes to next item

  • errorMessage参数，默认值：The requested content cannot be loaded. Please try again later. Error message when content can't be loaded
