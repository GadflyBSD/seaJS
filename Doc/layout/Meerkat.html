Meerkat 页面四方向提示插件

关于

Meerkat是一个jQuery插件用于在页面的底部和顶部创建一个固定的横幅。横幅的关闭有多种选项可以设置，可以一直不关闭直到离开当前页面，或者是当浏览器会话结束时自动关闭，可以设置时间定时关闭等。此外还可以设置横幅的背景、高度、显示/关闭动画的速度。

jQuery Meerkat 项目主页：http://meerkat.jarodtaylor.com

通过 data-option或者 data-config，设定参数

  • background参数，默认值：'none'，说明： The background option accepts CSS background (shorthand) property values.

  • opacity参数，默认值：null，说明： If opacity option is declared, meerkat will append an opacity layer, mirroring your background setting, but with transparency. This will not effect the opacity of the content, only the opacity background layer.

  • height参数，默认值：'auto'，说明： The height option accepts all CSS height property values (px, em, %, auto, etc).

  • width参数，默认值：'100%'，说明： The width option accepts all CSS width property values (px, em, %, auto, etc).

  • position参数，默认值：'bottom'，说明： The position setting will determine where Meerkat enters from and exits to. The available options are top, bottom, left, or right.

  • close参数，默认值：'.hide'，说明： The hide option attaches to an html element selector (ie. ".hide" or "#hide").

  • dontShowAgain参数，默认值：'#dont-show'，说明： The dontShowAgain option attaches to an html element selector (ie. ".dont-show" or "#dont-show"). Much like the hide option, this will remove Meerkat from the page, however, this also sets a cookie in the users browser telling it not to show Meerkat again. Depending on the cookieExpires setting, Meerkat will not show again for a set amount of days, or until the browser session ends (browser restarts).

  • cookieExpires参数，默认值：0，说明： If you use the dontShowAgain option, you can set the amount of days Meerkat will not show for. This option accepts whole numbers beginning with 0. Each number represents a day. If it's set to 0, Meerkat will not show again until that user's browser session has ended (browser restarts).'

  • removeCookie参数，默认值：'.removeCookie'，说明： By giving an HTML element a class or id that is assigned to the removeCookie option, users can click this element and the 'meerkat' cookie will be removed, allowing for meerkat to show again.

  • animationIn参数，默认值：'slide'，说明： This option accepts 'fade', 'slide', and 'none'. This will effect how Meerkat shows itself.

  • animationOut参数，默认值：'slide'，说明： This option accepts 'fade', 'slide', and 'none'. This will effect how Meerkat hides itself.

  • animationSpeed参数，默认值：'normal'，说明： The animationSpeed option accepts the following values: ("slow", "normal", or "fast") or the number of milliseconds to run the animation (e.g. 1000) When animationSpeed is set to a number; The number represents the duration of the sliding/fading animation (in milliseconds). The smaller the value, the faster the animation.

  • delay参数，默认值：0，说明： This option accepts a whole number representing seconds. Setting this option's value to '2' will delay Meerkat from showing for 2 seconds.

  • timer参数，默认值：Null，说明： This option accepts a whole number representing seconds. Setting this option's value to '5' will hide Meerkat automatically 5 seconds after it's appeared, without needing the user to click your hide element.

  • dontShowAgainAuto参数，默认值：False，说明： This option accepts true or false. This method works along side the timer method. If set to true, after Meerkat automatically closes, the dontShowAgainAuto will trigger the dontShowAgain method, which creates a cookie in the user's browser telling it not to show Meerkat again. Depending on the cookieExpires setting, Meerkat will not show again for a set amount of days, or until the browser session ends (browser restarts).

  • onMeerkatShow参数，默认值：function(){}，说明： The onMeerkatShow option is a callback function, an opportunity to add additional customization to Meerkat.

  • easingIn参数，默认值：'swing'，说明： The use of this option requires the jQuery Easing Plugin by George Smith. Coupled with the animationIn option, you have numerous choices for the easing of its animation. These options will not do anything without making reference to the jQuery Easing plugin BEFORE your jQuery Meerkat reference.

  • easingOut参数，默认值：'swing'，说明： The use of this option requires the jQuery Easing Plugin by George Smith. Coupled with the animationOut option, you have numerous choices for the easing of its animation. These options will not do anything without making reference to the jQuery Easing plugin BEFORE your jQuery Meerkat reference.
