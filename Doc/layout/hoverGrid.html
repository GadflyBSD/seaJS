图文鼠标悬停切换

通过 data-option或者 data-config，设定参数

  • itemClass参数：item类名。默认值：.item
  • width参数：item宽度。默认值：180px
  • height参数：item高度。默认值：180px
  • margin参数：item外间距。默认值：0 15px 0 0
  • padding参数：item内间距。默认值：15px
  • bgColor参数：背景颜色。默认值：#222
  • color参数：文字颜色。默认值：#fff
  • radius参数：圆角。默认值：8px
  • fadeInOut参数：淡入淡出效果。默认值：600
