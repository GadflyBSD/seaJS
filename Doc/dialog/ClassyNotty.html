ClassyNotty 提示框

关于

jQuery ClassyNotty 项目主页：http://www.class.pm/projects/jquery/classynotty/

通过 data-option或者 data-config，设定参数

  • title 参数，默认值：NULL，说明：弹框标题部分

  • content参数，默认值：NULL，说明：弹框正文内容

  • img 参数，默认值：NULL，说明：图片地址

  • showTime 参数，默认值：True，说明：是否显示弹框打开的时间

  • timeout 参数，默认值：0，说明：多长时间自动关闭，单位是毫秒，0为不自动关闭

  • hideMsg 参数，默认值：关闭，说明：关闭按钮的名称

  • top 参数，默认值：70px，说明：距离顶部的距离

  • offset 参数，默认值：20px，说明：距离两边（左边或者右边）的距离

  • width 参数，默认值：280px，说明：弹出框的宽度

  • position 参数，默认值：right，说明：弹出框显示位置，可选left/right

  • click 参数，默认值：function(){}，说明：点击关别按钮后的回调函数 

$.seaMethod('ClassyNotty', {
	title: '这是标题部分',
	content: '有标题和正文部分，带图片，不显示弹出框时间！',
	showTime: false,
	img: 'images/classynotty/thumb2.jpg'
});
