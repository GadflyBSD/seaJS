artDialog V6 弹框插件

关于

artDialog v6 —— 经典的网页对话框组件，内外皆用心雕琢。

兼容性测试通过：IE6~IE11、Chrome、Firefox、Safari、Opera

  1. 支持普通与气泡形式的对话框
  2. 信息无障碍支持：焦点管理；支持 ARIA 标准
  3. 面向未来：基于 HTML5 Dialog 的 API
  4. 支持标准与模态对话框
  5. 丰富的展示状态与友好的编程接口
  6. 自适应内容尺寸

jQuery artDialog 项目主页：http://aui.github.io/artDialog/

通过 data-option或者 data-config，设定参数

  • title参数，说明：写入对话框标题。 

    var d = dialog();
d.title('hello world');
d.show();

  • content参数，说明：写入对话框内容。参数支持String、HTMLElement类型。 

    //传入字符串：
var d = dialog();
d.content('hello world');
d.show();

//传入元素节点：
var elem = document.getElementById('test');
dialog({
    content: elem,
    id: 'EF893L'
}).show();

  • statusbar参数，说明：状态栏区域 HTML 代码，可以实现类似“不再提示”的复选框。注意：必须有按钮才会显示 

    var d = dialog({
    title: '欢迎',
    content: '欢迎使用 artDialog 对话框组件！',
    ok: function () {},
    statusbar: ''
});
d.show();

  • method参数，默认值：False，说明：对话框内容加载方式，可选tmpl（自动加载jQuery.tmpl.js）、ajax（使用Ajax框架页面显示一个地址）和False

  • tmpl参数，类型：Object，说明：当使用jQuery.tmpl.js时，所传递给模板的变量

  • ok参数，说明：确定按钮。回调函数this指向dialog对象，执行完毕默认关闭对话框，若返回 false 则阻止关闭。 

    dialog({
    ok: function () {
        this
        .title('消息')
        .content('hello world!')
        .width(130);
        return false;
    }
}).show();

  • okValue参数，默认值：ok，说明：确定按钮文本。 

    dialog({
    okValue: '猛击我',
    ok: function () {
        this.content('hello world!');
        return false;
    }
}).show();

  • cancel参数，默认值：False，说明：取消按钮。取消按钮也等同于标题栏的关闭按钮，若值为false则不显示关闭按钮。回调函数this指向dialog对象，执行完毕默认关闭对话框，若返回false则阻止关闭。 

    dialog({
    title: '消息',
    ok: function () {},
    cancel: function () {
        alert('取消');
    }
}).show();

  • cancelValue参数，默认值：cancel，说明：取消按钮文本。 

    dialog({
    cancelValue: '取消我',
    cancel: function () {
        alert('关闭');
    }
}).show();

  • button参数，类型：Array，说明：自定义按钮组。
    名称 类型 描述
    value String 按钮显示文本
    callback Function (可选) 回调函数this指向dialog对象，执行完毕默认关闭与销毁对话框（依次执行close()与remove()），若返回false则阻止关闭与销毁
    autofocus Boolean (默认值:false) 是否自动聚焦
    disabled Boolean (默认值: false) 是否禁用 

    dialog({
    button: [
        {
            value: '同意',
            callback: function () {
                this
                .content('你同意了');
                return false;
            },
            autofocus: true
        },
        {
            value: '不同意',
            callback: function () {
                alert('你不同意')
            }
        },
        {
            id: 'button-disabled',
            value: '无效按钮',
            disabled: true
        },
        {
            value: '关闭我'
        }
    ]
}).show();

  • width参数，说明：设置对话框宽度。 

    dialog({
    content: 'hello world'
})
.width(320)
.show();

  • height参数，说明：设置对话框高度。 

    dialog({
    content: 'hello world'
})
.height(320)
.show();

  • skin参数，说明：设置对话框额外的className参数。多个className请使用空格隔开。 

    dialog({
    skin: 'min-dialog tips'
}).show();

  • padding参数，默认值：继承 css 文件设置，说明：设置消息内容与消息容器的填充边距，即 style padding属性 

    dialog({
    content: 'hello world',
    padding: 0
}).show();

  • align参数，默认值：bottom left，说明：设置对话框与其他元素的对齐方式。如果show(elem)与showModal(elem)传入元素，align参数方可生效，支持如下对齐方式：

    • "top left"
    • "top"
    • "top right"
    • "right top"
    • "right"
    • "right bottom"
    • "bottom right"
    • "bottom"
    • "bottom left"
    • "left bottom"
    • "left"
    • "left top"
    var d = dialog({
    align: 'left',
    content: 'Hello World!',
    quickClose: true
});
d.show(document.getElementById('option-align'));

  • fixed参数，默认值：False，说明：开启固定定位。固定定位是 css2.1 position的一个属性，它能固定在浏览器某个地方，也不受滚动条拖动影响。IE6 与部分移动浏览器对其支持不好，内部会转成绝对定位。 

    dialog({
    fixed: true,
    title: '消息',
    content: '请拖动滚动条查看'
}).show();

  • quickClose参数，默认值：False，说明：是否点击空白出快速关闭。 

    var d = dialog({
    content: '点击空白处快速关闭',
    quickClose: true
});
d.show(document.getElementById('option-quickClose'));

  • autofocus参数，默认值：True，说明：是否支持自动聚焦

  • zIndex参数，默认值：1024，说明：重置全局zIndex初始值，用来改变对话框叠加高度。比如有时候配合外部浮动层 UI 组件，但是它们可能默认zIndex没有对话框高，导致无法浮动到对话框之上，这个时候你就可以给对话框指定一个较小的zIndex值。请注意这是一个会影响到全局的配置，后续出现的对话框叠加高度将重新按此累加。 

    dialog({
    zIndex: 87
}).show();

  • onshow参数，默认值：False，说明：对话框打开的事件。回调函数this指向dialog对象。 

    var d = dialog({
    content: 'loading..',
    onshow: function () {
        this.content('dialog ready');
    }
});
d.show();

  • onbeforeremove参数，默认值：False，说明：对话框销毁之前事件。回调函数this指向dialog对象。

  • onremove参数，默认值：False，说明：对话框销毁事件。回调函数this指向dialog对象。 

    var d = dialog({
    onclose: function () {
        alert('对话框已经关闭');
    },
    onremove: function () {
        alert('对话框已经销毁');
    },
    ok: function () {}
});
d.show();

  • onclose参数，说明：对话框关闭后执行的事件。回调函数this指向dialog对象。 

    var d = dialog({
    onclose: function () {
        alert('对话框已经关闭');
    },
    ok: function () {}
});
d.show();

  • onfocus参数，说明：聚焦对话框（置顶）。

  • onblur参数，说明：让对话框失去焦点。

  • onreset参数，说明：对话框位置重置事件。回调函数this指向dialog对象。

  • url参数，默认值：False，说明：

  • padding参数，默认值：False，说明：

  • id参数，默认值：False，说明：设定对话框唯一标识。1、可防止重复 ID 对话框弹出。2、 支持使用dialog.get(id)获取某个对话框的接口。 

    dialog({
    id: 'id-demo',
    content: '再次点击运行看看'
}).show();
dialog.get('id-demo').title('8888888888');

  • open参数，说明：判断对话框是否被打开。这是一个只读方法。

  • returnValue参数，说明：对话框返回值。 

    var d = dialog({
    title: '消息',
    content: '',
    ok: function () {
        var value = $('#property-returnValue-demo').val();
        this.close(value);
        this.remove();
    }
});
d.addEventListener('close', function () {
    alert(this.returnValue);
});
d.show();
