Ajax锚点异步加载效果

关于

在一些需要点击某个链接或按钮，另一个地方异步加载指定页面的地方，可以使用Ajax锚点异步加载效果，无刷新加载页面文件。
本文档就是使用了这个效果，点击左边的导航按钮后，右边的面板窗口自动异步加载相应的页面。
兼容IE 6, IE 7, IE 8, FF 3.6, Chrome 6.0, Safari 5.0, Opera 10.62。

通过HTML文档data属性

仅仅通过向页面元素添加data-sea="seaAnchor" 就可以为其赋予Ajax锚点异步加载效果。
以下是其他相关属性：

  • data-target 属性接受一个URL地址，将使用jQuery load函数Ajax加载url地址的内容，输出到指定位置；
  • data-anchorTo 属性接受一个选择器，所要输出的目标位置，默认是【#anchor-main】；
  • data-callback 属性接受一个回调函数的名称，默认是空，如指定了回调函数，需要定义该回调函数；

//custom_jAnchor回调函数
function custom_jAnchor(){
	$('[data-type=seaseaseaSnippet]').each(function(i, dom){
		$(dom).seaseaseaSnippetHighlight();
	});
}

//custom_collapse回调函数
function custom_collapse(){
	$('[data-type=collapse]', $('#anchor-main')).toggleCollapse();
}
