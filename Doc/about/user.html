seaJS 模块化 javascript 组件库【一般使用方法】

使用方法

给标签增加一个属性(data-sea="组件名")即可，比如需要调用日历组件，仅需要给input标签增加一个 data-sea="calendar" 的属性，日历就可以工作了。如下

<input type="text" data-sea="Calendar" name="inputDatepicker" placeholder="请输出年月日" />

使用 HTML 的自定义 data-* 属性进行一般性调用

  • JS调用方法： 
    <script type="text/javascript" src="/javascript/sea.js"></script>
<script type="text/javascript" src="/javascript/seajs-config.js"></script>
<script type="text/javascript">
seajs.use(['seajs-control', 'bootstrap', '其他需加载模块...'], function(){
	$(document).seaLoad();
});
</script>
  • data-sea属性：

    指定所需要调用的插件名称，例如： data-sea="Uploadify"将在该DOM上调用一个Uploadify上传组件。

  • data-option属性：

    接受一个Json数据格式或Object数据格式的插件参数，例如：

    • Json数据格式：data-option='{"param_1":"value_1","param_2":"value_2"..."param_n":"value_n"}'
    • Object数据格式：data-option="({param_1:'value_1',param_2:'value_2'...param_n:'value_n'})"
    • Object简写格式：data-option='param_1:'value_1',param_2:'value_2'...param_n:'value_n''，该格式适用于页面对 {} 指定为其他特殊用途时

  • data-config属性：

    接受一个对象名称作为插件的参数，当插件参数自定义部分较多，并有自定义回调函数的时候，使用该方法，例如： 

    <div data-sea="Plugin_1" data-config="Plugin_1_Option"></div>
<div data-sea="Plugin_2" data-config="Plugin_2_Option"></div>
<script type="text/javascript" >
var SeaOptions = ({
	Plugin_1_Option: {
		param_1: 'value_1',
		param_2: 'value_2',
		...
		param_n: 'value_n',
		callback: function(){
			some code ...
		}
	},
	Plugin_2_Option: {
		param_1: 'value_1',
		param_2: 'value_2',
		...
		param_n: 'value_n',
		callback: function(){
			some code ...
		}
	}
});
</script>

  • date-option和 date-config的优先顺序：
    • 如果没有指定date-option和 date-config将使用插件定义的默认值作为参数；
    • 如果既指定了date-option又指定了date-config，date-config将会覆盖date-option后的一个新的参数传递给插件；
    • 可以使用date-config定义多个调用的公共部分，使用date-option定义每个调用的不同部分；
    • 如果调用存在回调函数，最好使用date-config定义。
