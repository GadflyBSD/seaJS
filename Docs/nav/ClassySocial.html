ClassySocial 导航元素

关于

jQuery ClassySocial 项目主页：http://www.class.pm/projects/jquery/classysocial/

通过 data-option或者 data-config，设定参数

  • css参数，默认值：Nav/ClassySocial/css/jquery.classysocial.css，说明：CSS 样式表文件地址，如不加载请设定为False

  • event参数，默认值：click，说明：可选择的动作为click、hover

  • networks参数，说明：comma-separated list of social networks

  • orientation参数，默认值：arc，说明：开打方向，可选择：arc, line or linedown

  • arcStart参数，默认值：0，说明：start of the arc, from 1 to 360, default is 0

  • arcLength参数，默认值：180，说明：length of the arc, from 1 to 360

  • radius参数，默认值：80，说明：the radius of the arc

  • gap参数，默认值：50，说明：the gap (in pixels) between the profile bubbles

  • picture参数，默认值：seajs.data.base+'Nav/ClassySocial/images/share_core_square.jpg'，说明： can be either the name of the Facebook profile, an image url (ex images/test.jpg) or none,

  • imageType参数，默认值：picture，说明：can be facebook if you want your Facebook profile image to be shown by default or picture

  • theme参数，默认值：default，说明：
