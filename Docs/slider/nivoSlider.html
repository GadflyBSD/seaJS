nivoSlider 幻灯片

关于

Nivo Slider号称世界最棒的轻量级JQuery图片幻灯插件，按它的官网所说，“以漂亮和易于使用而闻名于世”！
jQuery nivo Slider项目主页：http://dev7studios.com/plugins/nivo-slider/ 版本号：jQuery Nivo Slider v3.2

通过 data-option或者 data-config，设定参数

  • css参数，属性：String 说明：所加载的Theme样式表地址，采用相对SeaJS的路径。如果不指定CSS样式表文件地址，则加载系统自带的Theme。

  • theme参数，属性：String，默认值：default 说明：加载系统自带的Theme，可选择：

    • light
    • dark
    • bar
    • default

  • easing参数，属性：Boolean，默认值：False 说明：切换图片时，图片说明部分是否自动使用动画效果

  • effect参数，属性：String，默认值：random 说明：过渡效果，默认是随机。可选：

    • sliceDown
    • sliceDownLeft
    • sliceUp
    • sliceUpLeft
    • sliceUpDown
    • sliceUpDownLeft
    • fold
    • fade
    • random
    • slideInRight
    • slideInLeft
    • boxRandom
    • boxRain
    • boxRainReverse
    • boxRainGrow
    • boxRainGrowReverse

  • slices参数，属性：Number，默认值：15 说明：切片效果时的数量

  • boxCols参数，属性：Number，默认值：8 说明：格子效果时的列数

  • boxRows参数，属性：Number，默认值：4 说明：格式效果时的行数

  • animSpeed参数，属性：Number，默认值：1000 说明：图片过渡时间

  • pauseTime参数，属性：Number，默认值：5000 说明：图片显示时间

  • startSlide参数，属性：Number，默认值：0 说明：从第几张图片开始（第一张为0）

  • directionNav参数，属性：Boolean，默认值：True 说明：是否显示图片方向切换按钮（上一页/下一页）

  • controlNav参数，属性：Boolean，默认值：True 说明：是否显示图片导航控制按钮（,2,3... ）

  • controlNavThumbs参数，属性：Boolean，默认值：False 说明：是否使用图片的缩略图做为导航控制按钮

  • pauseOnHover参数，属性：Boolean，默认值：True 说明：鼠标县浮时是否停止动画

  • manualAdvance参数，属性：Boolean，默认值：False 说明：是否手动切换

  • prevText参数，属性：String，默认值：Prev 说明：上一页方向切换按钮的显示文本

  • nextText参数，属性：String，默认值：Next 说明：下一页方向切换按钮的显示文本

  • randomStart参数，属性：Boolean，默认值：False 说明：开始图片是否随机

  • beforeChange参数，属性：Function，默认值：function(){} 说明：图片切换前触发函数

  • afterChange参数，属性：Function，默认值：function(){} 说明：图片切换后触发函数

  • slideshowEnd参数，属性：Function，默认值：function(){} 说明：在所有图片显示完毕后触发函数

  • lastSlide参数，属性：Function，默认值：function(){} 说明：在最后一张图片显示完毕后触发函数

  • afterLoad参数，属性：Function，默认值：function(){} 说明：图片加载完毕后触发函数

使用图片的缩略图做为导航控制按钮

使用图片的缩略图做为导航控制按钮，请先确保 controlNavThumbs被设置为 True。然后再图片上设置 data-thumb属性，值为缩略图的URL。如果URL错误，缩略图将不显示。例如： 

<div data-sea="seaNivoSlider">
    <img src="images/up.jpg" data-thumb="images/up_thumb.jpg" />
    <img src="images/monstersinc.jpg" data-thumb="images/monstersinc_thumb.jpg" />
    <img src="images/nemo.jpg" data-thumb="images/nemo_thumb.jpg" />
    <img src="images/walle.jpg" data-thumb="images/walle_thumb.jpg" />
</div>

为某个图片自定义过渡效果

您可以在需要指定过渡效果的图片上指定 data-transition属性即可，例如：

<img src="images/slide1.jpg" data-transition="slideInLeft" />
