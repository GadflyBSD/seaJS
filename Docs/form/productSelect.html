仿淘宝天猫的商品选择

关于

jQuery Camera 项目主页：http://www.pixedelic.com/plugins/camera

通过 data-option或者 data-config，设定参数

  • market参数，默认值：市场价，说明：市场价标签所显示的名称，当为 False时，不显示

  • present参数，默认值：当前价，说明：当前价标签所显示的名称，当为 False时，不显示

  • number参数，默认值：数量，说明：数量标签所显示的名称，当为 False时，不显示

  • numberInput参数，默认值：number，说明：表单数量输入框的名称

  • priceInput参数，默认值：price，说明：表单价格输入框的名称，隐藏框

  • propTextInput参数，默认值：propText，说明：表单产品属性输入框的名称，隐藏框

  • propertyInput参数，默认值：prop，说明：表单每个产品属性的值输入框的名称，隐藏框

  • css参数，默认值：Other/productSelect/jquery.productSelect.css，说明：样式表文件地址

HTML部分

<div data-sea="productSelect" data-config="goods"></div>

javaScript部分

var SeaOptions.goods = {
	property: {url:'http://www.cter1332.net/index/text', data:{type:'property'}},
	price:{url: 'http://www.cter1332.net/index/text', data:{type:'price'}}
}
