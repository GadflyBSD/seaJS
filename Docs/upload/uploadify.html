Uploadify 属性

  • auto
    布尔型
    设置auto为true，当文件被添加至上传队列时，将会自动上传。
  • buttonClass
    字符串
    为上传按钮添加类名。
  • buttonCursor
    字符串
    鼠标经过上传按钮时，鼠标的形状。可选值为‘hand’(手形) 和 ‘arrow’(箭头)。
  • buttonImage
    字符串
    定义“浏览”按钮背景图像的路径。给按钮设置背景图像的代码最好写在CSS文件中。
  • buttonText
    字符串
    定义显示在默认按钮上的文本。
  • checkExisting
    字符串
    定义检查目标文件夹中是否存在同名文件的脚本文件路径。
  • debug
    布尔型
    当其值为true时，开启SWFUpload调试模式。
  • fileObjName
    字符串
    定义上传数据处理文件中接收数据使用的文件对象名。
  • fileSizeLimit
    整型
    上传文件大小限制。
  • fileTypeDesc
    字符串
    可选择的文件类型的描述。此字符串出现在浏览文件对话框的文件类型下拉菜单中。
  • fileTypeExts
    字符串
    定义允许上传的文件后缀。
  • formData
    JSON对象
    定义在文件上传时需要一同提交的其他数据对象。
  • height
    整型
    上传按钮的高度（单位：像素）。
  • itemTemplate
    布尔型
    itemTemplate选项允许你为每一个添加到队列中选项设定一个不同HTML模板。
  • method
    字符串
    上传文件的提交方法，取值'post'或'get'。
  • multi
    布尔型
    设置值为false时，一次只能选中一个文件。
  • overrideEvents
    JSON
    该项定义了一组默认脚本中你不想执行的事件名称。
  • preventCaching
    布尔型
    如果设置为真，一个随机的值添加到SWF文件的URL，因此它不会缓存。
  • progressData
    字符串
    设置显示在上传进度条中的数据类型，可选项时百分比（percentage）或速度（speed）。
  • queueID
    字符串
    queueID选项允许你设置一个拥有唯一ID的DOM元素来作为显示上传队列的容器。
  • queueSizeLimit
    整型
    上传队列中一次可容纳的最大条数。
  • removeCompleted
    布尔型
    不设置该选项或者将其设置为false，将使上传队列中的项目始终显示于队列中，直到点击了关闭按钮或者队列被清空。
  • removeTimeout
    整型
    设置上传完成后从上传队列中移除的时间（单位：秒）。
  • requeueErrors
    布尔型
    设置为真时，上传队列重置或上传多次重试时，返回错误信息。
  • successTimeout
    整型
    表示uploadify的成功等待时间（单位：秒，默认30秒）。
  • swf
    字符串
    定义uploadify.swf的路径。
  • uploader
    字符串
    定义服务器端上传数据处理脚本文件uploadify.swf的路径。
  • uploadLimit
    整型
    定义允许的最大上传数量。
  • width
    整型
    定义浏览按钮的宽度。

Uploadify 事件

  • onCancel
    函数
    回调参数：file
    被取消的文件对象。
    设置onCancel选项，在文件上传被取消时，将允许运行一个自定义函数。
  • onClearQueue
    函数
    回调参数：queueItemCount
    被取消的队列的项目数量。
    设置onClearQueue选项，上传队列清空（激活ancel方法）时，将允许运行一个自定义函数。
  • onDestroy
    函数
    回调参数：
    该事件没有Arguments对象。
    销毁Uploadify实例（调用destroy方法）时触发该事件。
  • onDialogClose
    函数
    回调参数：queueData
    对象包括的队列信息包括：
    • filesSelected：在浏览文件对话框中选中的文件数量。
    • filesQueued：添加到队列中的文件数量。（没有返回错误）
    • filesReplaced：队列中替换文件的数量。
    • filesCancelled：从队列中取消的文件的数量。
    • filesErrored：返回错误信息的文件的数量。
    当浏览文件对话框关闭时触发该事件。如果该事件被添加到overrideEvents属性中，在添加文件到队列中发生错误时，将不会弹出默认错误信息。 
    function onDialogClose(queueData) {
    alert('选择的' + queueData.filesSelected + '个文件中有'+queueData.filesQueued + '个被添加到队列中。现在队列中一共有' + queueData.queueLength + '个文件。');
}
  • onDialogOpen
    函数
    回调参数：
    该事件没有Arguments对象。
    在浏览文件对话框被打开前一瞬触发该事件。
  • onDisable
    函数
    回调参数：
    该事件没有Arguments对象。
    调用disable方法禁用Uploadify实例时触发该事件。
  • onEnable
    函数
    回调参数：
    该事件没有Arguments对象。
    调用disable方法启用Uploadify实例时触发该事件。
  • onFallback
    函数
    回调参数：
    该事件没有Arguments对象。
    浏览器检测不到兼容版本的Flash时触发该事件。
  • onInit
    函数
    回调参数：instance
    uploadify 对象实例。
    调用Uploadify初始化结束时触发该事件。 
    function onInit(instance) {
	alert('队列ID是 ' + instance.settings.queueID);
}
  • onQueueComplete
    函数
    回调参数：queueData
    queueData对象包括的队列信息包括：
    • uploadsSuccessful：上传成功的文件数量。
    • uploadsErrored：上传失败的文件数量。
    队列中的所有文件被处理完成时触发该事件。 
    function onQueueComplete(queueData) {
	alert(queueData.uploadsSuccessful + ' 个文件上传成功.');
}
  • onSelect
    函数
    回调参数：file
    添加至上传队列的文件对象。
    每添加一个文件至上传队列时触发该事件。
  • onSelectError
    函数
    回调参数：file
    返回错误的文件对象。
    回调参数：errorCode
    返回的错误代码。在定义错误代码时可使用以下常量：
    • QUEUE_LIMIT_EXCEEDED：选定的文件数量超过了设置的限值
    • FILE_EXCEEDS_SIZE_LIMIT：文件的大小超过了设置的限值
    • ZERO_BYTE_FILE：文件大小
    • INVALID_FILETYPE：文件类型不在设置的范围内
    回调参数：errorMsg
    表示超过限值的错误信息。
    选择文件返回错误时触发该事件。每一个文件返回错误都会触发该事件。 
    function onSelectError(file, errorCode, errorMsg) {
	alert(file.name + '文件返回错误，没有添加到上传队列中.');
}
  • onSWFReady
    函数
    回调参数：
    该事件没有Arguments对象。
    当flash按钮载入完毕时触发该事件。
  • onUploadComplete
    函数
    回调参数：file
    上传完成的文件对象或返回的错误信息。
    每一个文件上传完成都会触发该事件，不管是上传成功还是上传失败。
  • onUploadError
    函数
    回调参数：file
    上传的文件对象。
    回调参数：errorCode
    返回的错误代码。
    回调参数：errorMsg
    返回的错误信息。
    回调参数：errorString
    错误详细信息（字符串）。
    上传失败时触发该事件。 
    function onUploadError(file, errorCode, errorMsg, errorString) {
	alert(file.name + ' 上传失败。详细信息: ' + errorString);
}
  • onUploadProgress
    函数
    回调参数：file
    上传文件对象。
    回调参数：bytesUploaded
    已上传的文件大小
    回调参数：bytesTotal
    文件的大小
    回调参数：totalBytesUploaded
    当前上传队列中，已上传的文件总大小。
    回调参数：totalBytesTotal
    当前上传队列中，上传文件的总大小。
    上传进度更新时触发该事件。 
    function onUploadProgress(file, bytesUploaded, bytesTotal, totalBytesUploaded, totalBytesTotal) {
	$('#progress').html('已上传'+totalBytesUploaded + ' 字节。共 ' + totalBytesTotal + ' 字节.');
}
  • onUploadStart
    函数
    回调参数：file
    上传的文件对象。
    在开始上传之前的瞬间会触发该事件。
  • onUploadSuccess
    函数
    回调参数：file
    上传成功的文件对象。
    回调参数：data
    上传数据处理文件（uploadify.php）输出的值。
    回调参数：response
    服务器端响应。上传成功为true，上传失败为false 。
    每一个文件上传成功时触发该事件。 
    function onUploadSuccess(file, data, response) {
	alert('文件 ' + file.name + ' 上传成功.详细信息： ' + response + ':' + data);
}

Uploadify 方法

  • cancel
    函数
    回调参数：fileID
    你想要取消上传的文件ID。需要取消多个文件时，添加的每个文件的ID作为参数。如果参数为星号“*”时，队列中的所有文件将被上传。如果没有ID，则取消队列中的第一个文件。
    回调参数：suppressEvent
    该项被设置为true时，onUploadCancel 事件不会被执行。一般在清空队列时使用。
    取消上传对象。 
    <input type="file" name="file_upload" id="file_upload" sea-toggle="Uploadify"/>
<a href="javascript:$('#file_upload').uploadify('cancel')">取消第一个上传文件</a> |
<a href="javascript:$('#file_upload').uploadify('cancel', '*')">清空队列</a> |
<a href="javascript:$('#file_upload').uploadify('upload', '*')">上传文件</a>
  • destroy
    函数
    回调参数：
    该方法没有Arguments对象。
    销毁Uploadify实例，并返回原文件域。 
    <input type="file" name="file_upload" id="file_upload" sea-toggle="Uploadify"/>
<a href="javascript:$('#file_upload').uploadify('destroy')">销毁Uploadify实例</a>
  • disable
    函数
    回调参数：setDisabled
    设置为true时禁用按钮，设置为false时，则按钮可用。
    控制浏览按钮是否可用。 
    <input type="file" name="file_upload" id="file_upload" sea-toggle="Uploadify"/>
<a href="javascript:$('#file_upload').uploadify('disable', true)">按钮不可用</a> |
<a href="javascript:$('#file_upload').uploadify('disable', false)">按钮可用</a>
  • settings
    函数
    回调参数：name
    你想获取或设置的名称。只有设置这个arguments对象才能返回值。
    回调参数：value
    你想要给setting使用的值。
    回调参数：resetObjects
    设置该参数，将更新value值。
    返回或更新一个Uploadify实例的值。 
    <input type="file" name="file_upload" id="file_upload" />
<a href="javascript:changeBtnText()">改变按钮文字</a> | <a href="javascript:returnBtnText();">显示按钮文字</a>
<script type="text/javascript">
    $(function() {
        $("#file_upload").uploadify({
            'swf'      : '/uploadify/uploadify.swf',
            'uploader' : '/uploadify/uploadify.php'
        });
    });
    function changeBtnText() {
        $('#file_upload').uploadify('settings','buttonText','BROWSE');
    }
    function returnBtnText() {
        alert('The button says ' + $('#file_upload').uploadify('settings','buttonText'));
    }
</script>
  • stop
    函数
    回调参数：
    该方法没有Arguments对象。
    停止当前上传。 
    <input type="file" name="file_upload" id="file_upload" data-sea="Uploadify" />
<a href="javascript:$('#file_upload').uploadify('upload', '*')">上传文件</a> |
<a href="javascript:$('#file_upload').uploadify('stop')">停止上传!</a>
  • upload
    函数
    回调参数：fileID
    需要上传的文件ID。需要上传多个文件时，添加的每个文件的ID作为参数。如果参数为星号“*”时，队列中的所有文件将被上传。
    上传指定文件或队列中的所有文件。 
    <input type="file" name="file_upload" id="file_upload"data-sea="Uploadify" />
<a href="javascript:$('#file_upload').uploadify('upload', '*')">上传文件</a>
