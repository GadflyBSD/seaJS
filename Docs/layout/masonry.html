Masonry 瀑布流

通过 data-option或者 data-config，设定参数

  • animationOptions参数，默认值：{ queue: false, duration: 500 }

    Options used for jQuery animation. See the jQuery API for animate options for details.

  • columnWidth参数，

    在布局中第一列的宽度，如果没有指定columnWidth，Masonry使用第一个item元素的宽度。建议如果布局中有多个列宽item元素时，请设置该参数。

  • gutterWidth参数，默认值：0

    添加列之间的额外间距。

  • isAnimated参数， Boolean，默认值：false

    在布局中是否使用jQuery animation动画效果

  • isFitWidth参数， Boolean，默认值：false

    If enabled, Masonry will size the width of the container to the nearest column. When enabled, Masonry will measure the width of the container’s parent element, not the width of the container. This option is ideal for centering Masonry layouts.Adds additional spacing between columns.

  • isResizable参数， Boolean，默认值：true

    当浏览器窗口大小改变时触发布局结构重新排列。

  • isRTL参数， Boolean，默认值：false

    启用从右到左的布局方式，以适应希伯来语和阿拉伯语的阅读方式

  • itemSelector参数， String，默认值：.box

    在布局中需要排列的选择器。

  • imagesLoaded参数， Boolean，默认值：false

    是否在所有图片加载完成后在进行瀑布流布局。

  • infinitescroll参数， Boolean，默认值：false

    是否当有多个页面的时候，异步加载infinitescroll插件，实现无限滚动效果。
